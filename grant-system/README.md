# Give N Go Grant Funding System

## Overview
Automated system to find, apply for, and win grants for the construction training program.

## The AI Team

| AI Employee | Role | What They Do |
|-------------|------|--------------|
| **Linda** | Legal Assistant | Grant applications, compliance, contracts, legal docs |
| **Stan** | Lead Sourcing | Finds grant opportunities, sends outreach, follows up |
| **Penny** | Content Creator | Presentations, case studies, blog posts, success stories |

## Workflow

### Phase 1: Grant Discovery (Stan)
- [ ] Set up Stan with grant database access
- [ ] Define keywords: workforce development, construction training, youth employment, vocational, pre-apprenticeship, Mecklenburg County, NC
- [ ] Build target list of:
  - Federal: DOL, DOE, DOJ grants
  - State: NC Department of Commerce, Workforce Development
  - Local: Mecklenburg County, Charlotte foundations
  - Corporate: Home Depot, Lowe's, construction associations

### Phase 2: Qualification (Stan + Linda)
- [ ] Stan scrapes/organizes: Deadline, Amount, Eligibility, Requirements
- [ ] Linda reviews for fit:
  - Does it match our population (court-referred youth)?
  - Is the funding amount worth the effort?
  - Do we meet all eligibility requirements?

### Phase 3: Application Building (Linda + Penny)
**Template Package (reuse across grants):**

1. **Program Description** (reusable)
   - PACT model overview
   - Target population
   - Expected outcomes

2. **Problem Statement** (from JCPC — already done)
   - Local data: 59.9% youth associate with delinquent peers
   - 77.1% have school problems
   - 37.5% have family criminal records

3. **Solution Narrative** (reusable)
   - Pre-apprenticeship certificate training
   - 25% classroom / 75% hands-on work
   - Evidence-based curricula: LST, Guiding Good Choices

4. **Budget Template**
   - Personnel (Program Manager, Instructors)
   - Equipment (tools, safety gear)
   - Facility (workshop space)
   - Curriculum costs

5. ** outcomes Metrics**
   - Youth served
   - Completion rate
   - Job placement rate
   - Recidivism reduction

### Phase 4: Submission Tracking (Stan)
- [ ] Calendar deadlines
- [ ] Submission reminders
- [ ] Follow-up on pending applications

### Phase 5: Stewardship (Penny + Linda)
- [ ] Progress reports for funders
- [ ] Success stories (photos, testimonials)
- [ ] Thank you letters / recognition

## Winning Grant Elements (from JCPC)

Based on the JCPC application, successful grants include:

1. **Clear problem statement with data**
   - Use Mecklenburg County statistics
   - Reference 2016-17 Needs Assessment

2. **Evidence-based approach**
   - PACT (US DOL recognized)
   - Life Skills Training (LST)
   - Guiding Good Choices

3. **Specific outcomes**
   - 30 youth capacity
   - 90-day program
   - Job placement track

4. **Strong budget justification**
   - $13,073 per youth (from JCPC)
   - Detailed line items
